## 🚀 GitHub Pages'e Yükleme Adımları

### 1. Yeni Repo Oluştur
- GitHub'da sağ üstten **New repository** tıkla
- İsim ver (örn. `inkılap-tarihi`)
- **Public** seç → **Create repository**

### 2. Dosyaları Yükle
Bu klasördeki dosyaları repoya yükle:
- `index.html` ← Ana uygulama dosyası
- `README.md` ← Bu dosya

**Yükleme yöntemleri:**
- GitHub web arayüzünde **Add file → Upload files** ile sürükle-bırak
- Ya da Git CLI:
  ```bash
  git init
  git add .
  git commit -m "İlk yükleme"
  git remote add origin https://github.com/KULLANICI/REPO.git
  git push -u origin main
  ```

### 3. GitHub Pages'i Etkinleştir
- Repo sayfasında **Settings** → **Pages**
- **Source:** `Deploy from a branch`
- **Branch:** `main` → `/ (root)` → **Save**
- 2-3 dakika bekle, ardından `https://KULLANICI.github.io/REPO/` adresine git

### 4. Telefona Ekle (İsteğe Bağlı)
Mobil tarayıcıda uygulamayı açtıktan sonra:
- **iPhone:** Safari → Paylaş → "Ana Ekrana Ekle"
- **Android:** Chrome → Menü (⋮) → "Ana ekrana ekle"

Uygulama, telefonda tam ekran simge olarak görünür! 📱
